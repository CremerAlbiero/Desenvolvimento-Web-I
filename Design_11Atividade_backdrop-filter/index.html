Backdrop-filter

Exemplos visuais de como você pode usar o backdrop-filter para criar efeitos visuais sobre o plano de fundo.

Efeito Blur

O efeito 'blur' cria um efeito de desfoque no plano de fundo.

Pode ser um efeito visual para destacar um elemento sobre outro, muito utilizado para sobrepor e desfocar imagens.

Exemplo atual: 'backdrop-filter: blur(10px)'

Efeito Brightness

A Proriedade 'brightness()' é usada para ajustar o brilho do elemento no plano de fundo, é possível aumentar ou diminuir.

Por padrão, o brilho é (1). Assim, qualquer valor acima disso está aumentando o brilho.

Exemplo atual: 'backdrop-filter: brightness(1.2)'.

Efeito Brightness

Também pode ser utilizado para diminuir o brilho, aplicando um valor menor que 1.

Exemplo atual: 'brightness(0.2)'

O brilho foi diminuido para 20% do normal, tornando o elemento de fundo mais escuro.

Efeito Contrast

A propriedade 'contrast()' é usada para ajustar o contraste do plano de fundo.

Torna as cores mais ou menos acentuadas.

Considera APENAS as cores do plano de fundo.

Exemplo: 'backdrop-filter: contrast(2)'. As cores estão 2 vezes mais acentuadas.

Efeito Grayscale

A propriedade 'grayscale()' é usada para converter o conteúdo em tons de cinza.

Cria um efeito visual em que o elemento de fundo e seu conteúdo são exibidos em escala de cinza, sem cores.

Os valores variam de '0' a '1'. Portanto, usar grayscale(1) já é suficiente.

Exemplo atual: 'backdrop-filter: grayscale(1)'

Efeito Sépia

O efeito é utlizado para aplicar um efeito convertendo suas cores em tons sépias.

Isso cria um efeito visual que lembra as fotografias antigas.

Exemplo atual: 'backdrop-filter: sepia(1)'

Efeito Sépia

O efeito é utlizado para aplicar um efeito convertendo suas cores em tons sépias.

Isso cria um efeito visual que lembra as fotografias antigas.

Exemplo atual: 'backdrop-filter: sepia(1)'

Efeito hue-rotate

A propriedade hue-rotate() em CSS é usada para ajustar a matiz (ou cor) de um elemento, permitindo que você crie uma rotação de cores em torno do espectro de cores.

Essa propriedade permite que você aplique uma transformação na matiz do elemento, mudando as cores, mas mantendo a luminosidade e a saturação.

Exemplo atual: 'backdrop-filter: hue-rotate(90deg). Rotacionando as cores em 90 graus.
